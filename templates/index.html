Bem-vindo à API do ComparadorIA

Esta API permite que você compare configurações de redes de computadores de forma automatizada utilizando inteligência artificial.

Documentação da API

  • Endpoint de Comparação: /api/compare/
  • Método: POST
  • Autenticação: Token (via header Authorization: Token <seu_token>)
  • Formato do JSON: 
    {
    "instruction": { ... },
    "instructor_config": { ... },
    "instructor_network": { ... },
    "student_config": { ... },
    "student_network": { ... }
}
